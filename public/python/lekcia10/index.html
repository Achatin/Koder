Pygame Zero

Pygame je populárna knižnica na tvorbu počítačových hier. Pygame Zero je zjednodušená verzia originálu. Je to teda jednoduchý game-engine 🎮. Na jeho použitie ti stačia doterajšie znalosti a trochu vysvetliviek.

Inštalácia Pygame Zero

Pygame sa inštaluje cez nástroj pip. Pip je inštalátor cudzích knižníc na Python. Dokáže nainštalovať akúkoľvek potrebnú knižnicu jedným príkazom:

pip install pgzero

⚠️ Ty si však pip musíš najprv nainštalovať. To urobíš týmito príkazmi v termináli:

curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py

python3 get-pip.py

Hra v Pygame Zero

Aby sme si overili, že inštalácia prebehla úspešne, vyskúšame si spustiť prázdny súbor.

Príkazom:

pgzrun game.py

Vykreslenie

Ako prvý krok si vykreslíme pozadie hry. Použijeme špeciálnu funkciu draw(), ktorá na to slúži.

WIDTH = 300
HEIGHT = 300

def draw():
    screen.fill((128, 0, 0))

Ako ďalší krok si vytvoríme actora ‘Alien’. Dáme mu tento obrázok.

🎈 Pomôcka: Obrázok si môžeš stiahnuť pravým kliknutím a uložením.

Obrázok si ulož do priečinku “images”.

alien = Actor('alien')
alien.pos = 100, 56

WIDTH = 500
HEIGHT = alien.height + 20

def draw():
    screen.clear()
    alien.draw()

Pohyb

Na rozpohybovanie aliena môžeš vyskúšať príkaz:

alien.topright = 10, 10

Určite si si všimol, že sa niekde pohol. Ten pohyb bol ale instantný a pre nás neviditeľný. Preto pridáme špeciálnu funkciu update(), aby sa pohyboval pomaly smerom doprava.

def update():
    alien.left += 2
    # ked vyjde z obrazovku, vrátime ho na zaciatok
    if alien.left > WIDTH:
        alien.right = 0

Klikanie

Opäť použijeme špeciálnu funkciu on_mouse_down(), ktorá kontroluje klikanie.

def on_mouse_down(pos):
    if alien.collidepoint(pos):
        print("Eek!")
    else:
        print("You missed me!")

Teraz si stiahni ďalší obrázok aj zvukovú nahrávku, ktorú použijeme v hre:

alien_hurt.png
eep.wav

Súbory si porozdeľuj do priečinkov nasledovne:

.
├── images/
│   └── alien.png
│   └── alien_hurt.png
├── sounds/
│   └── eep.wav
└── game.py

Uprav si hru a použi nové súbory.

def on_mouse_down(pos):
   if alien.collidepoint(pos):
       sounds.eep.play()
       alien.image = 'alien_hurt'

Ako si si asi všimol, alien sa po kliknutí zmení na nový obrázok, ale nikdy sa nevráti do pôvodného stavu. Hru si trochu poupravíme.

def on_mouse_down(pos):
   if alien.collidepoint(pos):
       set_alien_hurt()

def set_alien_hurt():
   alien.image = 'alien_hurt'
   sounds.eep.play()

def set_alien_normal():
   alien.image = 'alien'

def set_alien_hurt():
   alien.image = 'alien_hurt'
   sounds.eep.play()
   clock.schedule_unique(set_alien_normal, 1.0)
