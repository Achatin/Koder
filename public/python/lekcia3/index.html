Zdravím! 👋

Na dnešnej lekcii sa naučíš používať podmienky. Táta kľúčová súčasť programovania ti dovolí doplniť si ku hre Hero Progression 🧙‍♂️ nové možnosti.

Opakovanie

🔰 Úloha: Vytvorte herný obchod. Do obchodu vypíšte, koľko má hráč peňazí. Vytvorte aspoň jednu položku a vymyslite jej cenu.

Príklad obchodu

Welcome to the Shop 👋
You have 90 💸
----------------------
🎁 Gift - 10 💸

Odsadenie kódu

Odsadenie kódu alebo indentácia je odsek v kóde. V Pythone sú tieto odseky veľmi dôležité, lebo určujú, čo všetko patrí do podmienky alebo funkcie.

Podmienky

Na skontrolovanie stavu nejakej situácie používame v programovaní podmienky. Kľúčovým slovíčkom je if (ak) a za ním nasleduje situácia/podmienka, ktorú chceme skontrolovať. Tento riadok ukončíme dvojbodkou. Do nového riadku po odsadení píšeme potom všetko, čo sa má stať, ak platí podmienka.

# ak premenna pet sa rovna "unicorn", vypis "🦄"
if pet == "unicorn":
    # indentacia kodu
    print("🦄")

⚠️ Nesprávne použitie

if online == True:
# chyba indentacia, teda print nepatri ku podmienke :(
print("Player is Online 🟢")

Zložené podmienky

Niekedy je užitočné použiť aj opak podmienky - else (inak). Ak podmienka if neplatí, vykoná sa else zátvorka.

online = False

# vsimni si indentaciu
if online == True:
    print("Player is online ✅")
else:
    print("Player is offline ❌")

Vytvoriť vieme aj viacnásobnú podmienku, pomocou ktorej vieme zahrnúť viacero prípadov. Po jednoduchej if podmienke nasleduje elif, potom znenie podmienky a nakoniec po dvojbodke obsah (čo sa má odohrať po splnení). Na koniec viacnásobnej podmienky vieme rovnako pripojiť aj else podmienku.

if pet == "unicorn":
    print("🦄")
elif pet == "dragon":
    print("🐲")
elif pet == "python":
    print("🐍")
else:
    print("no pet ❌")

Porovnávacie znamienka

Porovnávacie znamienka používame pri porovnávaní javov. Využívajú sa v podmienkach a cykloch.

1. == rovná sa

level = 10
if level == 10:
    print("😎 PRO")

2. != nerovná sa

level = 6
if level != 10:
    print("🤓 Noob")

3. > väčši ako

exp = 10

# 11+, nemáme dostatok exp
if exp > 10:
    print("Level Up! 🎉")
else:
    print("Potrebuješ aspoň 11 exp!")

4. >= väčšie rovné

exp = 10

// 10+, získame nový level
if exp >= 10:
    print("Level Up! 🎉")
else:
    print("Potrebuješ aspoň 10 exp!")

5. < menšie ako

hp = 10

if hp < 10:
    print("Your hero is weak! Health Potion 🍯")
else:
    print("Healthy hero 💪")

6. <= menšie rovné

hp = 10

if hp <= 10:
    print("Your hero is weak! Health Potion 🍯")
else:
    print("Healthy hero 💪")

🔰 Úloha: Vytvorte mechanizmus, ktorý zistí, či je hrdina nažive. Použite podmienku na skontrolovanie životov. Vypíšte, či je hrdina nažive alebo nie.

🔰 Úloha: Využite herný obchod z minulej hodiny a vytvorte pre neho podmienku, ktorá zistí, či máme dostatok peňazí na nákup z obchodu. Výsledok vypíšte.

🔰 Úloha: Naprogramujte podmienku tak, aby nahradila číslicu za zodpovedajúce emoji. Využite pri tom else if. Príklad: 7 => 7️⃣

Logické operátory

and - A zároveň Podmienka platí, keď všetky výrazy sú pravdivé.

online = True
playing = False

# ak je hráč online A ZÁROVEŇ nie je v hre
if online == true and playing == false:
    print("Ready to play 🎮")

or - Alebo Podmienka platí, keď aspoň jeden výraz je pravdivý.

pet = "🦊"
# ak máme peta 🐲 ALEBO 🦊
if pet == "🐲" or pet == "🦊":
    print("Pet used 🔥")
