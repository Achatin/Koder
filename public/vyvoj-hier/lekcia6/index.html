Dnes pridáme animácie a checkpoints do našej hry a tým dokončíme celý Galaxy Rush 🌌. Na záver tohto projektu si ukážeme ako hru zverejniť na internete a zahrať si ju hoci aj na mobile.

Animácie

Animácie v hrách fungujú na jednoduchom princípe - rapídneho menenia obrázkov. Ak chceš vytvoriť animáciu pre pohyb hráča, potrebuješ pravidelne meniť jeho sprite.

function animation()
    --na striedacku menime vsetky sprites hraca
    if player.sprite < 3 then
        player.sprite += 1
    else
        --vratenie sa na povodny sprite
        player.sprite = 1
    end
end

Funkciu potom môžeš zavolať pri pohybe hráča - vo funkcii movement().

Všimni si, že animácia prebieha príliš rýchlo. Rýchlosť animovania vieš meniť pomocou časovača ⌚.

Časovač

V Pico-8 existuje funkcia time(), ktorá meria čas od jej použitia. Využitím tejto funkcie vieme vytvoriť časovač.

Na animácie vytvoríme v tabuľke player premennú anim = time().

Teraz vieme našu funkciu animation() upraviť pridaním časovača.

function animation()
    --ak casovac presiahne 0.1 sekundy (cas snimku animacie)
    if (time() - player.anim) > 0.1 then
        --aktualizacia casovaca
        player.anim = time()
        --na striedacku menime vsetky sprites hraca
        if player.sprite < 3 then
            player.sprite += 1
        else
            --vratenie sa na povodny sprite
            player.sprite = 1
        end
    end
end
